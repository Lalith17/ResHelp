## âš™ï¸ Getting Started

1. **Clone the Repo**

   ```bash
   git clone https://github.com/lalith17/reshelp.git
   cd reshelp
   ```

2. **Install Backend**

   ```bash
   cd backend
   npm install
   ```

3. **Install Frontend**

   ```bash
   cd ../frontend
   npm install
   ```

4. **Configure Environment Variables**
   Add `.env` files in `frontend` and `backend` with:

   - MongoDB URI
   - OAuth credentials
   - AI/Resume API keys (if applicable)

5. **Run the App**
   Backend:

   ```bash
   cd backend
   npm run dev
   ```

   Frontend:

   ```bash
   cd ../frontend
   npm start
   ```
